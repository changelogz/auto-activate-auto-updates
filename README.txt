Simple Utility Plugin

Activate as either plugin or MU Plugin will then automatically turn on auto updates for plugins when they are activated turning WordPress Auto Updaters opt out not opt-in

#Change Log

v1.2.0 - Added storing state when a plugin is deactivated 
v1.1.0 - Added Auto Updating for preexisting plugins, if none have already been added to the auto update system.
